MACHINE LEARNING

Spine Segmentation

Orthopaedic and Traumatology Research Lab in the University of Hong Kong - 2018

Project Description:

Using machine learning to make segmentation of vertebrae in the spine easier. A part of a larger project to diagnose bone disorders in the lumbar spine.

Responsibilities:

Collect data: download mRI scans from medical records at Queen Mary Hospital; edit data: make sure dimensions are all the same throughout data; create masks: use 3D Slicer to create masks/labels for the mRI scans; clean data: delete broken files, hidden files, and blurry images; organize data: make sure dataset follows the coco style format; train MaskRCNN with this new data.Through these steps I was able to understand the details of every step of the machine learning process in this sort of project.

Resources Used:

  • Used 3D slicer and Growcut Algorithm to create masks
  • Mask_RCNN Github Repository

Skills learned:

  • Researching skills
  • Data collecting
  • Script writing
  • Advancement in skills in Python

I was able to not only learn the process of machine learning but how it can be applied to medical research and practices as well. By tactfully combining technology and medical research, the lab is able to achieve greater feats and create applications to assist doctors in the field.

Chest Disorder Diagnoses

Prince of Wales Hospital - 2018

Project Description:

Improve accuracy and efficiency in diagnoses by using a machine to diagnose chest disorders from X-Rays.

Responsibilities:

Edit data: crop and resize images so that they can be read by the program; organize data: sort the X-Rays into categories based on disorder labels for flow usage; train/refine Resnet50 with data

Resources Used:

  • ResNet50
  • FineTuning ResNet50 Github Repository

Skills learned:

  • How to train a model from start to finish and how to improve accuracy
  • develop stronger skills with Python and became more familiar with the language

I was able to not only learn the process of machine learning but how it can be applied to medical research and practices as well. By tactfully combining technology and medical research, the lab is able to achieve greater feats and create applications to assist doctors in the field.
